# Local Messenger - Конспект проекта

## Общая информация
**Название проекта:** Local Messenger  
**Тип проекта:** Локальный мессенджер для образовательных учреждений  
**Статус:** Рабочий прототип  
**Тип:** Клиент-серверное приложение  

## Технологии
### Backend
- FastAPI (Python)
- SQLite (база данных)
- JWT (аутентификация)
- WebSocket (real-time)

### Frontend
- PyQt5 (интерфейс)
- Python (программирование)
- HTTP/REST (запросы)
- WebSocket (соединения)

## Функционал
### Реализовано
1. Регистрация и вход
2. Отправка текстовых сообщений
3. Отправка изображений
4. Real-time обновления
5. Групповые чаты
6. История сообщений
7. Удаление сообщений
8. Поиск по чатам
9. Онлайн-статусы

### Архитектура
Клиент → Сервер → База данных

#### Клиент
- Интерфейс PyQt5
- Сетевые запросы
- Локальное кэширование

#### Сервер
- Обработка запросов
- Управление WebSocket
- Работа с базой данных

#### База данных
- SQLite
- Таблицы: users, messages, chats

## Установка
### Требования
- Python 3.8+
- Библиотеки: FastAPI, PyQt5, SQLite

### Запуск
1. Установить зависимости
2. Запустить сервер
3. Запустить клиент

## Ограничения
- Работа в локальной сети
- Базовый функционал
- Только изображения в base64
